# SeperateLetters

## Problem

Write a program that manipulates a String. The String should be called sentence. The program should prints each of the characters in the String on a separate line 

### Sample Input

I drink tea

### Sample Output

I <br>
d <br>
r <br>
i <br>
n <br>
k <br>
t <br>
e <br>
a <br>
